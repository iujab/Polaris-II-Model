A real-time atmospheric dispersion model based on Gaussian solutions. Metrology datasets downloaded from: RTMA, ALOS30, and SRTM90 . Open source, MIT licensed, and available on GitHub.

Features
  • Mobile-based atmopheric dispersion model with server-based computation solutions
  • GeoJSON, metrology & signal processing Solutions with build-in model formatter
  • Logical multiple layer marker clustering via the leaflet marker cluster plugin
  • Gaussian dispersion model with Euler based plume centerline using typeahead.js
  • Timestep accuracy evaluation with NMSE and FB from Fukushima datasets
  • Inbuilt Pasquill-Gifford Stability classes and grouped layer control plugin
