A real-time atmospheric dispersion model based on Gaussian solutions. Metrology datasets downloaded from: RTMA, ECMWF Integrated Forecast System (IFS), and SRTM90 . Open source, available on GitHub.

Features
  • Mobile-based atmopheric dispersion model with server side computation solutions
  • GeoJSON, metrology & signal processing solutions with built-in model formatter PolarisCOMET
  • Logical multiple layer marker clustering via the leaflet marker cluster plugin
  • Gaussian dispersion model with Lagrange particle path based plume centerline using SciPython
  • Timestep accuracy evaluation with NMSE and FB from Fukushima datasets
  • Inbuilt Pasquill-Gifford Stability classes and grouped layer control plugin

The data provided on this site is for informational and planning purposes only.

Absolutely no accuracy or completeness guarantee is implied or intended. All information generated are subject to such variations and corrections as might result from true deposition and/or accurate field survey.
